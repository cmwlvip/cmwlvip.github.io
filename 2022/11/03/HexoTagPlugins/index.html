Include Posts

包括指向其他帖子的链接。

使用此标记时，可以忽略永久链接和文件夹信息，例如语言和日期。
例如: {% post_link my %}。
只要帖子的文件名 my.md，这将起作用,即使帖子位于Source/posts/2015-02-my-family-holiday，并且具有永久链接2018/en/my。
您可以自定义要显示的文本，而不是显示帖子的标题。不支持在 Markdown 语法 []() 中使用post_path。
默认情况下，帖子的标题和自定义文本会被转义。您可以使用转义选项禁用转义。
例如：
显示贴子的标题
{% post_link TheCharmOfMarkdown %}

了不起的 Markdown

显示自定义文本
{% post_link TheCharmOfMarkdown 'Markdown 的魅力' %}

Markdown 的魅力

(escape)转义标题
{% post_link HexoTagPlugins 'How to use tag in title' %}

How to use <b> tag in title

不转义标题
{% post_link HexoTagPlugins 'bold custom title' false %}

bold custom title
